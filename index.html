CAD Autograder

Welcome to the CAD Autograder, developed by students from Duke University. This program allows you to compare student DXF files against a fully correct one and produce a grade for each student.
This autograder can check for the following errors:
-Incorrect, missing, or extra lines, circles, ellipses, arcs, or hatching
-Incorrect scale
-Incorret orientation (rotation error)
-Incorrect lineweight or color

This Autograder grades each entity individually, sums the grades and divides by the total number of entities to produce a final student grade.
-Dropdown menus to give suggested grading options, but you can also type in specific values.
-For extra entities, select how many points out of 100 you want to be subtracted for each extra entity.
-Color, lineweight, and hatching errors are specific to an entity. For these errors, select what percentage of each entity you want subtracted. (for example, if you enter "10" for lineweight error, a line with correct dimensions but incorrect lineweight will be considered a 90% correct entity)
-Scaling and rotation penalties are applied and subtracted from the final grade at the end.
-Click on the 'Start grading!' button when all information has been input. Results will be outputted in a text file.
